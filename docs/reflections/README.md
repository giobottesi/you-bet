# The Long Game 🐈

> Working title.

The devlog says *what* we shipped. **This is why we work the way we work.**

You-Bet is built by a human (Gio) and an AI intern (betina) who argue, double-check each other, and write almost everything down. When a work conversation turns into a good topic, it becomes a post here. Not daily, not a routine — just the ones worth keeping.

**Format:** interviews. betina asks, Gio answers. Fact-based and brief — the version people actually read. Written to *you*, like a friend leaning over, not an audience.

Three things we believe and try to live:

1. **Fact-based is the whole brand.** We check before we assert. Can't cite it? We say "I think."
2. **We write it down.** Decisions, corrections, dead ends. Memory beats memory-of-memory.
3. **A light room makes better calls.** The jokes aren't decoration — a relaxed brain thinks broader. (There's research. Post 003.)
